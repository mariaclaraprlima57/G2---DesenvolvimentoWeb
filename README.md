# Echo - Rede Social

Projeto desenvolvido para a disciplina de Desenvolvimento de Software para Web.

Uma rede social inspirada no Twitter, permitindo autenticação de usuários, publicação de posts e sistema de curtidas.

## 🛠 Tecnologias Utilizadas

### Front-end

- React
- Vite
- React Router DOM
- Axios

### Back-end

- Node.js
- Express

### Banco de Dados

- SQLite3

### Autenticação

- JWT (JSON Web Token)
- Bcrypt

## 🚀 Como executar

### Backend

Entre na pasta

```bash
cd backend
```

Instale as dependências

```bash
npm install
```

Execute

```bash
npm run dev
```

Servidor disponível em

```
http://localhost:3000
```

---

### Frontend

Entre na pasta

```bash
cd frontend
```

Instale as dependências

```bash
npm install
```

Execute

```bash
npm run dev
```

Aplicação disponível em

```
http://localhost:5173
```

## Funcionalidades

### Autenticação

- Cadastro de usuário
- Login
- Logout
- Senha criptografada com Bcrypt

---

### Posts

- Visualizar posts
- Criar posts
- Listagem pública

---

### Curtidas

- Curtir posts
- Descurtir posts

---

### Usuário não autenticado

Pode:

- Visualizar todos os posts

---

### Usuário autenticado

Pode:

- Publicar novos posts
- Curtir
- Descurtir
- Fazer logout
